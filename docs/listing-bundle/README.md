# Listing Bundle Metadata Templates

This directory contains templates for preparing metadata required for exchange listings and token information.

## Required Files

### Token Metadata (token-metadata-template.json)
Use this template to create your token metadata JSON file. Replace placeholder values with actual information.

### Contract Addresses (contract-addresses.md)
Document all deployed contract addresses for different networks.

### Audit Reports (audit-reports.md)
List all security audit reports and their locations.

### Logo Assets
Place logo files (PNG, SVG) in this directory. Recommended sizes:
- 256x256px for main logo
- 64x64px for favicon
- SVG for scalable use

## Usage

1. Copy the template files
2. Fill in actual values
3. Validate JSON syntax
4. Compress the entire bundle for submission
