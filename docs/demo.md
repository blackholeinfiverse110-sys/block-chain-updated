# Demo Recording Instructions

## Overview

This document provides placeholder instructions for recording a demo of the Blackhole Blockchain mainnet deployment.

## Prerequisites

- Ensure all deployment files are prepared as per `docs/mainnet-deploy.md`
- Have access to recording software (e.g., OBS Studio, Camtasia)
- Prepare test environment with sample data

## Recording Steps

1. **Setup Environment**
   - Start with a clean system
   - Load the prepared configuration files
   - Initialize the local node storage in `data/` directory

2. **Record Deployment Process**
   - Follow the step-by-step instructions in `docs/mainnet-deploy.md`
   - Narrate each step clearly
   - Highlight key configuration points

3. **Showcase Features**
   - Demonstrate bridge functionality
   - Show transaction processing
   - Display monitoring dashboards

4. **Post-Processing**
   - Edit the recording for clarity
   - Add captions and annotations
   - Export in multiple formats (MP4, WebM)

## Tips

- Speak clearly and at a moderate pace
- Pause briefly between major steps
- Use screen highlighting for important elements
- Test audio and video quality before final recording

- [ ] Environment prepared
- [ ] Scripts tested
- [ ] Recording software configured
- [ ] Backup plan in place
