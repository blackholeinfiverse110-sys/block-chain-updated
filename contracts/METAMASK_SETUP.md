# 🦊 MetaMask Setup Guide for BHX Token

## Step 1: Add Polygon Amoy Testnet

### Method 1: Automatic Addition (Recommended)
1. **Visit**: https://chainlist.org/
2. **Search**: "Amoy"  
3. **Click**: "Add to MetaMask" for Polygon Amoy Testnet
4. **Approve**: In MetaMask popup

### Method 2: Manual Addition
1. **Open MetaMask**
2. **Click**: Network dropdown (top)
3. **Select**: "Add Network"
4. **Enter these details**:
   ```
   Network Name: Polygon Amoy Testnet
   New RPC URL: https://rpc-amoy.polygon.technology/
   Chain ID: 80002
   Currency Symbol: MATIC
   Block Explorer URL: https://amoy.polygonscan.com/
   ```
5. **Save**

## Step 2: Add BHX Token

1. **Switch to Polygon Amoy** network in MetaMask
2. **Click**: "Import tokens" (bottom of assets list)
3. **Select**: "Custom token" tab
4. **Enter**:
   ```
   Token contract address: 0x64Dd946E7425898FE8FaB6D36eA050071190be40
   Token symbol: BHX
   Token decimal: 18
   ```
5. **Click**: "Add Custom Token"
6. **Click**: "Import Tokens"
